# Node mailer instant code



### Prerequisites
* node >=0.12.8
* npm ^3.10.0

## Quick Start:

```shell

# Browse to project folder
cd /path/to/project/this/folder

# Install dependencies
npm install

# add all congig details in config.json
# start node 
node index.js
```
